# 🔍 Manjaro & Arch Linux — Home Search

PT: Uma página para simplificar as buscas entre as ferramentas 
úteis do Manjaro e ArchLinux, centralizando o acesso à 
documentação oficial em um único lugar.

EN: A page to simplify searches between the useful tools of 
Manjaro and ArchLinux, centralizing access to official 
documentation in one place.

## 💡 Motivação

A documentação do Manjaro e Arch Linux é rica, mas dispersa 
em múltiplas plataformas. Este projeto nasceu da necessidade 
de ter um ponto único de busca, evitando navegar entre 
wiki, fórum e outras fontes separadamente.
